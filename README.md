# royal-reflections.inn
the royal reflections inn Website
Project Overview
The royal reflections inn is a responsive static website developed using HTML5, CSS3, and JavaScript. It provides visitors with information about the hotel, available rooms, facilities, photo gallery, and an online room booking form. The website is designed to be user-friendly, responsive, and SEO-friendly.
